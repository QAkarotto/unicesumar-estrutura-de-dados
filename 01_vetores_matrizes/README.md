# Vetores e Matrizes em C

Módulo com exemplos progressivos de vetores, matrizes e strings em C.

## Arquivos de Exemplo

| Arquivo | Conceito Principal |
|---------|--------------------|
| [01_vetores_basicos.c](01_vetores_basicos.c) | Declaração, inicialização e acesso a vetores |
| [02_operacoes_vetores.c](02_operacoes_vetores.c) | Busca, soma, maior e menor elemento |
| [03_matrizes_basicas.c](03_matrizes_basicas.c) | Declaração, inicialização e acesso a matrizes |
| [04_operacoes_matrizes.c](04_operacoes_matrizes.c) | Soma, multiplicação escalar e operações matriciais |
| [05_strings.c](05_strings.c) | Manipulação de strings como vetor de char |

## Objetivos de Aprendizagem

- Declarar e percorrer vetores e matrizes corretamente
- Aplicar operações comuns em estruturas lineares e bidimensionais
- Trabalhar com strings em C de forma segura e clara

## Compilação e Execução

```bash
gcc -Wall -Wextra -std=c11 arquivo.c -o programa
```

```bash
./programa
```

# Ponteiros em C

Módulo dedicado aos fundamentos de ponteiros e alocação dinâmica de memória em C.

## Arquivos de Exemplo

| Arquivo | Conceito Principal |
|---------|--------------------|
| [01_ponteiros_fundamentos.c](01_ponteiros_fundamentos.c) | Endereço de memória, desreferenciação e NULL |
| [02_aritmetica_ponteiros_vetores.c](02_aritmetica_ponteiros_vetores.c) | Aritmética de ponteiros em vetores |
| [03_aritmetica_ponteiros_matrizes.c](03_aritmetica_ponteiros_matrizes.c) | Aritmética de ponteiros em matrizes |
| [04_alocacao_dinamica_memoria.c](04_alocacao_dinamica_memoria.c) | Alocação dinâmica com malloc, calloc e realloc |
| [05_estruturas_dinamicas.c](05_estruturas_dinamicas.c) | Vetores e matrizes dinâmicas |

## Objetivos de Aprendizagem

- Entender o modelo de memória com ponteiros
- Utilizar passagem por referência em funções
- Aplicar aritmética de ponteiros em vetores e matrizes
- Alocar e liberar memória dinâmica com boas práticas

## Compilação e Execução

```bash
gcc -Wall -Wextra -std=c11 arquivo.c -o programa
```

```bash
./programa
```
